# Dependencies

* **Minecraft 1.21.7**

### Required Mods:
* **Meteor Client (1.21.7)**

* **Fabric Loader**

* **Fabric API**

### Installation:
1. Install Fabric Loader for Minecraft 1.21.7.
2. Place mods JAR in .minecraft/mods folder
3. Launch Minecraft with Fabric profile
4. Enable module in Meteor Client menu
